# Task Planner

Веб-приложение для управления проектами, задачами и тегами. Проект построен на архитектуре раздельных Frontend (React) и Backend (Django REST Framework) сервисов.

## Технологический стек
* **Backend:** Python 3.11, Django 5.x, DRF, PostgreSQL 17.
* **Frontend:** React 18, JavaScript, Axios, Vite, Nginx.
* **Инфраструктура:** Docker, Docker Compose, OpenAPI (Swagger).

## Основной функционал
* **Управление задачами:** Создание, редактирование, логическое удаление, выполнение (статус "Готово"). Валидация дедлайнов (диапазон от сегодня до 10 лет вперед).
* **Управление проектами:** Логическое удаление, редактирование названия и описания, персональные дашборды для каждого проекта.
* **Теги:** Уникальные для каждого пользователя, динамическое создание/удаление, связь «многие ко многим» с задачами.
* **UX/UI:** Адаптивная верстка, переключение тем (светлая/темная), навигация с сохранением истории переходов.
* **Безопасность:** Изоляция данных пользователей на уровне БД, Token-аутентификация, защита от XSS через экранирование React.

## Варианты запуска

### Вариант 1: Docker (Рекомендуемый)
Используется для развертывания системы целиком с настроенной базой данных.
1. Убедитесь, что Docker Desktop запущен, а локальная служба PostgreSQL остановлена (порт 5432 должен быть свободен).
2. Выполните в корне проекта:
   ```bash
   docker compose up --build
   ```
3. Приложение доступно на `http://localhost`, API документация на `http://localhost:8000/api/docs/`.

### Вариант 2: Локальный запуск (Разработка)
Используется для внесения изменений в код.
1. **Backend:**
   ```bash
   cd backend
   venv\Scripts\activate
   python manage.py runserver
   ```
2. **Frontend:**
   ```bash
   cd frontend
   npm run dev -- --force --host
   ```

## Тестирование
Для проверки корректности бизнес-логики бэкенда реализован набор Unit-тестов.
* **Запуск тестов:** 
  ```bash
  docker compose exec backend python manage.py test
  ```
* **Проверка покрытия кода:** 
  ```bash
  docker compose exec backend coverage run --source='tasks' manage.py test
  docker compose exec backend coverage report
  ```
Текущий уровень покрытия составляет 95%.
